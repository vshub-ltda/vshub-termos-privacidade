Política de Privacidade

Última atualização: Abril de 2026

Índice

1. Informações Gerais

A VS\CHAT está comprometida em proteger a privacidade dos usuários de sua plataforma de automação de conversas e atendimento digital.

A VS\CHAT utiliza a API oficial do WhatsApp Business, garantindo conformidade com os requisitos técnicos e operacionais da Meta.

2. Coleta de Informações

Dados da Conta:

  • Nome completo
  • Endereço de e-mail
  • Informações de pagamento (quando aplicável)

Dados de Integrações:

Ao conectar o WhatsApp (oficial ou não oficial), poderão ser coletados:

  • Tokens de acesso e credenciais de API
  • Dados de perfil e configuração
  • Metadados de conversas

Dados de Automação:

  • Mensagens processadas
  • Histórico de conversas
  • Dados de contatos e leads
  • Configurações de fluxos e chatbot

3. Uso das Informações

Os dados coletados são utilizados para:

  • Prestação dos serviços contratados
  • Execução de automações
  • Processamento de pagamentos
  • Suporte técnico
  • Geração de relatórios e análises
  • Melhoria contínua da plataforma
  • Cumprimento de obrigações legais

4. Integração com Inteligência Artificial

A VS\CHAT poderá integrar serviços de inteligência artificial de terceiros.

O tratamento de dados seguirá as políticas dos respectivos fornecedores.

5. Compartilhamento de Dados

A VS\CHAT poderá compartilhar dados com:

  • Provedores de tecnologia e infraestrutura
  • APIs de mensageria (como WhatsApp)
  • Integrações autorizadas pelo usuário
  • Autoridades legais, quando exigido
A VS\CHAT não vende dados pessoais sob nenhuma hipótese.

6. Segurança da Informação

A VS\CHAT adota medidas técnicas e organizacionais compatíveis com boas práticas de mercado, incluindo:

  • Criptografia de dados
  • Controle de acesso
  • Monitoramento de segurança
  • Backups periódicos

Incidentes de Segurança

Em caso de incidente:

  • Usuários afetados poderão ser notificados em até 72 horas, conforme a LGPD
  • Medidas corretivas serão adotadas imediatamente

📧 Reporte: contato@vshub.com.br

7. Direitos dos Usuários

Nos termos da Lei Geral de Proteção de Dados (LGPD), o usuário pode:

  • Acessar seus dados
  • Corrigir dados incorretos
  • Solicitar exclusão
  • Solicitar portabilidade
  • Revogar consentimento

A VS\CHAT poderá solicitar verificação de identidade antes de atender requisições.

8. Exclusão de Dados

O usuário pode solicitar exclusão de seus dados a qualquer momento:

📧 contato@vshub.com.br

Prazos:
  • Processamento da solicitação em até 30 dias
  • Dados em backup removidos conforme ciclo técnico

A exclusão é irreversível.

9. Integração com WhatsApp

A VS\CHAT utiliza a API oficial do WhatsApp Business fornecida pela Meta.

Dados coletados incluem:

  • Mensagens enviadas e recebidas
  • Números de telefone
  • Metadados de comunicação

Uso dos dados:

  • Automação de atendimento
  • Comunicação com clientes
  • Relatórios e análises
O usuário é responsável por obter consentimento dos contatos.

10. Proibições e Compromissos

A VS\CHAT NÃO:

  • Vende ou comercializa dados
  • Utiliza dados para vigilância
  • Cria perfis discriminatórios
  • Utiliza dados para análise de crédito ou emprego

Todos os dados são utilizados exclusivamente para prestação dos serviços.

11. Conformidade Legal

A VS\CHAT atua em conformidade com a Lei Geral de Proteção de Dados (LGPD) e demais normas aplicáveis.

12. Transferência Internacional de Dados

A VS\CHAT poderá utilizar serviços de infraestrutura e tecnologia localizados fora do Brasil.

Nesses casos, são adotadas medidas adequadas para garantir a proteção dos dados conforme exigido pela LGPD.

13. Planos e Pagamentos

A VS\CHAT opera em modelo de assinatura.

  • Não há garantia de reembolso após utilização do serviço
  • Valores e condições podem ser alterados a qualquer momento
